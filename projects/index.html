# 项目总体介绍

先腾开源的项目可分三个类别：

  1. 支持类项目; 这类项目基本都是通用算法、工具或者解决方案，对先腾的框架没有依赖。它们通常作为开源的通用算法的补充。
  2. 框架类项目; 总体设计详细描述了框架的设计思想、技术路线和产品的组成。
  3. 工具与服务类项目; 这部分为独立的服务模块，参见通用模块说明。
